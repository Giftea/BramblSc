A Wallet manages your Keys, which will allow you to create and prove transactions. This section covers how to initialize and use a wallet.

## Prerequisites

The following requires access to a Wallet API instance. You can initialize once using the following snippet:

```scala
val walletApi = WalletApi.make[F](walletKeyApi)
```

Where:
- `F` is a type parameter bound to a context parameter of type `Async[F]`. 
- `walletKeyApi` is an implementation of the <ScaladocLink path="co/topl/brambl/dataApi/WalletKeyApiAlgebra.html"><code>WalletKeyApiAlgebra</code></ScaladocLink>.

For more information, refer to the `make` function's <ScaladocLink path="co/topl/brambl/wallet/WalletApi$.html#make[F[_]](WalletKeyApiAlgebra[F])(Async[F]):WalletApi[F]"><code>signature</code></ScaladocLink>.

### Example: Initializing with Cats Effect `IO`

If youâ€™re using Cats Effect `IO` and the default implementation of <ScaladocLink path="co/topl/brambl/servicekit/index.html"><code>ServiceKit</code></ScaladocLink>, initialize the `WalletApi` like this:

```scala
import cats.effect.IO
import co.topl.brambl.servicekit.WalletKeyApi
import co.topl.brambl.wallet.WalletApi

val walletApi = WalletApi.make[IO](WalletKeyApi.make())
```

This snippet sets up an `IO`-based Wallet API instance using `WalletKeyApi`.
